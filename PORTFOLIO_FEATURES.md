# Portfolio Features & Capabilities

## 🎯 **Dual-Mode Portfolio System**

This portfolio supports **two viewing modes** that users can toggle between:

### **1. Multi-Page Mode** (Default)
- Traditional multi-page setup with navigation via URL routing
- Each section has its own dedicated page:
  - `/` - Home
  - `/about` - About Section
  - `/experience` - Internships & Experience  
  - `/projects` - Featured Projects
  - `/certifications` - Certifications
  - `/publications` - Publications & Patents
  - `/contact` - Contact Information

**Best for:** Traditional portfolio browsing, deep dives into specific sections

### **2. Single-Page Mode**
- Modern single-page scrollable experience
- All content in one seamless scroll
- Features smooth scroll navigation with sections:
  - Home
  - About
  - Journey (Timeline & Background)
  - Achievements
  - Experience
  - Skills
  - Projects
  - Certifications
  - Publications
  - Contact

**Best for:** Quick overview, impressive demo, modern feel

## ✨ **Interactive Features**

### **Mode Toggle (Navbar)**
- Located in the top-right navbar
- Visual toggle switch showing current mode
- Animated transition between modes
- Persistent mode selection (saved in localStorage)

### **View Mode Switcher (Animation)**
- Animated notification when mode changes
- Shows which mode is now active
- Auto-scrolls to top on mode switch

### **Smooth Scroll Navigator (Left Sidebar - Single Page Mode)**
- Visible on desktop screens
- Shows current section being viewed
- Navigate to any section with one click
- Up/Down buttons for sequential navigation
- Automatic updates as you scroll

### **Quick Preview Modal (Bottom-Right)**
- Available in Single-Page mode only
- Zap/Lightning icon (+) button
- Grid of all sections with descriptions
- One-click navigation to any section
- Modal overlay with instant close

### **Command Palette (Cmd/Ctrl + K)**
- Universal navigation tool (works in both modes)
- Search through sections and commands
- Switch between Single/Multi-page mode
- Copy current URL to clipboard
- Keyboard shortcuts displayed
- Clean dark interface with gradient accents

## 🚀 **How to Use**

### **Switch Modes**
1. Click the toggle switch in the navbar (top-right)
2. Or use Command Palette: `Cmd/Ctrl + K`, search "Switch"

### **Navigate in Single-Page Mode**
- Use left sidebar (desktop only)
- Click Quick Preview button (bottom-right)
- Use Command Palette (Cmd/Ctrl + K)
- Or scroll naturally

### **Navigate in Multi-Page Mode**
- Click nav items in navbar
- Use browser back/forward
- Direct URL entry
